# Документация по Play %PLAY_VERSION% 

> Play - это высоко производительный фреймворк для Java и Scala веб приложений, который интегрирует компоненты и API, который вам неоходим для разработки современных веб приложений. 
>
> Play базируется на легковесной, stateless, дружественной к вебу архитектуре и возможностях предсказуемого и минимального потребления ресурсов (CPU, память, потоки) для высокомасштабируемых приложений благодаря своей реактивной модели, базирующейся на Iteratee IO.
