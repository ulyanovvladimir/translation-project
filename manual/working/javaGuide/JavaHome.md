# Play для Java разработчиков

Java API для разработчиков Play приложений доступно в пакете `play`. 

> API доступное в пакете `play.api` (такое как `play.api.mvc`) зарезервиновано для Scala разработчиков. В качестве Java разработчика, см. `play.mvc`.
