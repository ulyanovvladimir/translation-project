# Play для Java разработчиков

Java API для разработчиков приложений на Play доступно в пакете `play`. 

> API доступное в пакете `play.api` (такое как `play.api.mvc`) зарезервировано для Scala разработчиков. В качестве Java разработчика, см. `play.mvc`.
